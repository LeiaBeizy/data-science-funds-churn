# data-science-funds-churn
# 🇱🇺 Ciência de Dados para Fundos de Investimento: Previsão de Churn

Este projeto desenvolve uma solução analítica para identificar investidores em risco de retirar capital (*churn*) num fundo de investimento baseado no Luxemburgo,
cruzando dados de CRM (Perfil) e Transações (Comportamento).

## 💼 O Problema de Negócio
Gestores de fundos enfrentam dificuldades em monitorizar manualmente milhares de contas. 
Clientes que realizam resgates sem um contacto comercial ativo representam um risco crítico de perda de Ativos Sob Gestão (AUM).

## 🛠️ Solução Desenvolvida
Utilizando **Python (Pandas e Matplotlib)**, o script realiza:
1. Fusão de dados de perfil (CRM) e histórico financeiro recente.
2. Criação de uma métrica automatizada de risco baseada em regras de negócio.
3. Quantificação do impacto financeiro total em risco.
4. Geração de relatórios visuais para suporte à decisão da equipa de marketing e vendas.

## 📊 Resultados do Protótipo
* Capital Identificado em Risco Crítico: 100,000 EUR (Cliente: Maria Silva).
* Ação Recomendada: Disparar alerta imediato no CRM para o gestor de conta reatar contacto.
